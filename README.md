## ✨ 功能特点

1. **可视化运势**: 结合金融技术分析，用股票 K 线图展示 1-100 岁的人生运势起伏，直观呈现人生“牛市”与“熊市”。
2. **AI 深度批断**: 利用 Gemini 3 Pro 强大的推理能力，生成性格、事业、财富、婚姻、健康、六亲及发展风水等多维度报告。
3. **发展风水**: 提供详尽的方位建议、地理环境选择（如沿海、山区）及开运布局。
4. **Web3 特供**: 专为加密货币交易者设计的“币圈交易运势”板块，包含暴富流年预测与适合的交易风格建议。

## 🚀 快速开始

### 1. 安装依赖

```bash
npm install
# 或者
pnpm install
# 或者
yarn install
```

### 2. 启动开发服务器

```bash
npm run dev
```

打开浏览器访问 `http://localhost:5173` 即可使用。

### 3. 构建生产版本

```bash
npm run build
```

## 🛠️ 技术栈

- **前端框架**: React 19 + Vite
- **UI 样式**: TailwindCSS
- **图表库**: Recharts
- **AI 模型**: Google Gemini API (`gemini-3-pro-preview`)
